Baseball 2018

I'm going to try and follow the baseball season using a python app. It's not much to look at right now, but I'll add to it gradually. The idea is to run a script once a day and update this website.
I'm starting with the pybaseball package by James LeDoux and going from there. The goal is to learn more python and more about baseball statistics. All my work can be found here.
