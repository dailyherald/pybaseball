I'm going to try and follow the baseball season using a python app. It's not much to look at right now, but I'll add to it gradually. The idea is to run a script once a day and update this website.
I'm starting with the pybaseball package by James LeDoux and going from there. The goal is to learn more python and more about baseball statistics. All my work can be found here.


Updated Wednesday, April 4
I really ought to automate that date. While not reflected in the page below, I have updated things behind the scenes. First, in the github repository, all the code to this is in the freeze.py script as well as the playing jupyter notebook.
Next, I've added batting and pitching stats for each team to the standings file. Pybaseball gets these from fangraphs.com. I'm going to add some graphics below the standings, likely based on some work I've done with Seaborn. I'm thinking of running regressions on several things and watch how they change as the season progresses.
I'll also set up a simple home run bar chart to track teams' progress. I believe it's projected a couple of teams may break the single-season record.
After this page is done, I'll start figuring out what kind of analytics I want to do for the Cubs and Sox, as well as a head-to-head comparison page.
